# Channel 类型 - Channel Types

[← 返回数据类型](../)

Channel（通道）是 Go 中的引用类型，用于在 goroutine 之间传递数据。

## <Badge text="Channel 基础" type="tip" />

### 什么是 Channel

Channel 是一个类型安全的管道，用于在 goroutine 之间传递数据。

```go
// 声明 channel 类型
var ch chan int      // 声明 int 类型的 channel
ch = make(chan int)  // 创建 channel

// 简写
ch := make(chan string)  // 创建 string 类型的 channel
```

<Badge text="重要" type="danger" /> Channel 是<strong>引用类型</strong>，包含指向底层队列的指针。

### Channel 方向

```go
// 双向 channel（默认）
var ch1 chan int

// 只发送 channel
var ch2 chan<- int

// 只接收 channel
var ch3 <-chan int
```

## <Badge text="创建 Channel" type="info" />

### 无缓冲 Channel

```go
ch := make(chan int)
// 发送：ch <- value（阻塞，直到有接收者）
// 接收：value := <-ch（阻塞，直到有发送者）
```

<Callout type="tip" title={<Badge text="同步保证" type="tip" />}>
  无缓冲 channel 提供<strong>同步保证</strong>：发送和接收必须同时发生。
</Callout>

### 缓冲 Channel

```go
ch := make(chan int, 10)  // 容量为 10
// 发送：ch <- value（缓冲未满时不阻塞）
// 接收：value := <-ch（缓冲为空时阻塞）
```

<Callout type="warning" title={<Badge text="容量选择" type="warning" />}>
  <strong>缓冲容量选择</strong>

  • 无缓冲：需要强同步，确保数据被处理<br/>
  • 有缓冲：异步处理，提高吞吐量<br/>
  • 容量建议：2 的幂次方（2, 4, 8, 16...）
</Callout>

## <Badge text="发送和接收" type="warning" />

### 基本操作

```go
ch := make(chan string)

// 发送
ch <- "hello"

// 接收
msg := <-ch

// 接收并检查 channel 是否关闭
msg, ok := <-ch
if !ok {
    // channel 已关闭
}
```

### range 遍历

```go
ch := make(chan int, 10)

// 发送方
go func() {
    for i := 0; i < 5; i++ {
        ch <- i
    }
    close(ch)  // 发送完毕关闭 channel
}()

// 接收方
for value := range ch {
    fmt.Println(value)
}
// 当 channel 关闭且数据取完后自动退出循环
```

## <Badge text="关闭 Channel" type="warning" />

### 关闭规则

```go
ch := make(chan int)

// 关闭 channel（只有发送方应该关闭）
close(ch)

// ❌ 不要重复关闭
// close(ch)  // panic: close of closed channel
```

<Callout type="danger" title={<Badge text="重要" type="danger" />}>
  <strong>关闭注意事项</strong>

  • 只有<strong>发送方</strong>应该关闭 channel<br/>
  • 不要向已关闭的 channel 发送（panic）<br/>
  • 不要关闭已关闭的 channel（panic）<br/>
  • 关闭 nil channel 会 panic
</Callout>

## <Badge text="Channel 是引用类型" type="info" />

### 引用行为

```go
ch1 := make(chan int)
ch2 := ch1  // ch2 和 ch1 指向同一个 channel

// 发送到 ch2
ch2 <- 42

// 从 ch1 接收
value := <-ch1  // value = 42
```

### nil Channel

```go
var ch chan int  // nil channel

// ch <- 1     // 永久阻塞
// value := <-ch  // 永久阻塞

// nil channel 的特性：
// - 发送会永久阻塞
// - 接收会永久阻塞
// - close(ch) 会 panic
```

<Badge text="零值用途" type="tip" /> nil channel 可用于<strong>禁用 select 分支</strong>：

```go
select {
case <-ch:
    fmt.Println("received")
case <-nil:  // 永久阻塞，相当于禁用
    fmt.Println("never")
}
```

## <Badge text="多 Channel 操作" type="danger" />

### Select

```go
ch1 := make(chan string)
ch2 := make(chan int)

select {
case msg := <-ch1:
    fmt.Println("从 ch1 接收:", msg)
case num := <-ch2:
    fmt.Println("从 ch2 接收:", num)
default:
    fmt.Println("没有 channel 可用")
}
```

<Badge text="默认分支" type="info" /> `default` 分支在<strong>没有 channel 就绪时执行</strong>。

## <Badge text="Channel 比较" type="warning" />

```go
ch1 := make(chan int)
ch2 := make(chan int)
ch2 <- 42

// channel 只能比较是否为 nil
if ch1 == nil {
    fmt.Println("ch1 是 nil")
}

// ❌ 不能直接比较两个 channel
// if ch1 == ch2  // 编译错误
```

## 练习

1. 创建 channel 并传递数据

<details>
<summary>查看答案</summary>

```go
package main

import "fmt"

func main() {
    // 创建 string 类型的 channel
    ch := make(chan string)

    // 启动 goroutine 发送数据
    go func() {
        ch <- "Hello, Channel!"
    }()

    // 接收数据
    msg := <-ch
    fmt.Println("收到:", msg)
}
```

**解释**：展示了 channel 的基本用法。
</details>

2. 使用 range 遍历 channel

<details>
<summary>查看答案</summary>

```go
package main

import "fmt"

func main() {
    ch := make(chan int, 3)

    // 发送数据
    go func() {
        for i := 1; i <= 3; i++ {
            ch <- i
        }
        close(ch)
    }()

    // range 遍历
    for value := range ch {
        fmt.Println(value)
    }
}
```

**解释**：展示了 range 自动检测 channel 关闭。
</details>

3. 使用 select 等待多个 channel

<details>
<summary>查看答案</summary>

```go
package main

import (
    "fmt"
    "time"
)

func main() {
    ch1 := make(chan string)
    ch2 := make(chan int)

    go func() {
        time.Sleep(100 * time.Millisecond)
        ch1 <- "来自 ch1"
    }()

    go func() {
        time.Sleep(50 * time.Millisecond)
        ch2 <- 42
    }()

    select {
    case msg := <-ch1:
        fmt.Println("收到:", msg)
    case num := <-ch2:
        fmt.Println("收到:", num)
    }
}
```

**解释**：展示了 select 多路复用，先收到的先处理。
</details>
