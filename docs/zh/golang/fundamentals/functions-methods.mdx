## 快速预览

### 函数基础

```go
package main

import "fmt"

// 无参数无返回值
func greet() {
    fmt.Println("Hello, World!")
}

// 带参数和返回值
func add(a, b int) int {
    return a + b
}

// 多返回值（Go 特色）
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

func main() {
    greet()
    result := add(5, 3)
    fmt.Println(result)  // 8

    q, err := divide(10, 3)
    if err != nil {
        fmt.Println("错误:", err)
        return
    }
    fmt.Println("商:", q)
}
```

### 方法基础

```go
package main

import "fmt"

type Rectangle struct {
    width  float64
    height float64
}

// 值接收者方法
func (r Rectangle) Area() float64 {
    return r.width * r.height
}

// 指针接收者方法
func (r *Rectangle) Scale(factor float64) {
    r.width *= factor
    r.height *= factor
}

func main() {
    rect := Rectangle{width: 3, height: 4}
    fmt.Println("面积:", rect.Area())  // 12

    rect.Scale(2)
    fmt.Printf("缩放后: %.1f x %.1f\n", rect.width, rect.height)
    // 输出: 缩放后: 6.0 x 8.0
}
```

## 核心概念速览

### 函数 vs 方法

| 特性 | 函数 | 方法 |
|-----|------|------|
| 定义 | `func name()` | `func (recv T) name()` |
| 接收者 | 无 | 值或指针 |
| 调用 | `name()` | `recv.name()` |
| 用途 | 独立功能 | 类型行为 |

### 关键特性

<Badge text="多返回值" type="info" />
Go 的多返回值常用于错误处理：

```go
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

<Badge text="闭包" type="warning" />
函数可以捕获外部变量：

```go
func makeCounter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}
```

<Badge text="Defer" type="warning" />
确保函数返回前执行代码：

```go
file, err := os.Open(path)
if err != nil {
    return err
}
defer file.Close()  // 函数返回前自动关闭
```

## 常见错误

<Callout type="danger" title="注意！">
  避免以下常见错误：

  1. **忽略错误返回值** - 始终检查 error
  2. **闭包循环变量** - Go 1.22+ 已修复
  3. **Defer 在循环中** - defer 不会立即执行
  4. **指针接收者混用** - 保持一致性
  5. **Panic 用于正常错误** - 应该返回 error
</Callout>
