## 📚 模块详解

### 🔧 核心工具 (Core Utilities)

#### must.go - 错误断言

简化错误处理的工具，提倡快速失败策略：

```go
import "github.com/lazygophers/utils"

// 传统 Go 错误处理
data, err := risky.Operation()
if err != nil {
    return nil, fmt.Errorf("operation failed: %w", err)
}

// LazyGophers 方式 - 更简洁、更快速的开发
data := utils.Must(risky.Operation()) // 出错时 panic

// 检查并处理错误
utils.MustSuccess(saveData(data))

// 检查布尔返回值
value := utils.MustOk(getValue()) // 如果第二个返回值为 false 则 panic
```

**何时使用**：
- 应用初始化阶段
- 配置加载
- 测试代码
- 不应失败的错误路径

#### orm.go - 数据库操作

简化的数据库操作接口：

```go

type User struct {
```

### ⏰ 时间与调度 (Time & Scheduling)

#### xtime/ - 高级时间处理

支持农历、生肖、节气的现代化时间库：

```go
import "github.com/lazygophers/utils/xtime"

func timeExample() {
    cal := xtime.NowCalendar()

    // 公历日期
    fmt.Printf("日期: %s\n", cal.Format("2006-01-02"))

    // 农历日期
    fmt.Printf("农历: %s\n", cal.LunarDate())           // "农历二零二三年六月廿九"
    fmt.Printf("生肖: %s\n", cal.Animal())              // "兔 (Rabbit)"
    fmt.Printf("节气: %s\n", cal.CurrentSolarTerm())   // "处暑 (End of Heat)"

    // 节气判断
    if cal.IsSolarTerm() {
        fmt.Println("今天是节气日")
    }

    // 干支纪年
    stemBranch := cal.StemBranch()  // "癸卯"
}
```

### 数据库操作

```go
type User struct {
    ID    int64  `json:"id"`
    Name  string `json:"name" validate:"required"`
    Email string `json:"email" validate:"required,email"`
    Age   int    `json:"age" default:"0" validate:"min=0,max=150"`
}

func SaveUser(db *sql.DB, user *User) error {
    // 验证结构体
    if err := utils.Validate(user); err != nil {
        return err
    }

    // 转换为数据库格式
    data, err := utils.Value(user)
    if err != nil {
        return err
    }

    // 保存到数据库
    _, err = db.Exec("INSERT INTO users (data) VALUES (?)", data)
    return err
}
```

### 并发处理

```go
func processingExample() {
    // 创建工作池
    pool := routine.NewPool(10)
    defer pool.Close()

    // 提交任务并使用熔断器保护
    for i := 0; i < 100; i++ {
        taskID := i
        pool.Submit(func() {
            // 熔断器保护防止故障
            result := hystrix.Do("process-task", func() (interface{}, error) {
                return processTask(taskID)
            })

            fmt.Printf("任务 %d 结果: %v\n", taskID, result)
        })
    }

    // 等待完成（带超时）
    wait.For(5*time.Second, func() bool {
        return pool.Running() == 0
    })
}
```
