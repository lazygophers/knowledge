# Array Type

[← 返回数据类型](../)

数组是<strong>固定长度</strong>的相同类型元素序列。

## <Badge text="数组基础" type="tip" />

### 数组声明

```go
// 声明数组，元素为零值
var arr1 [5]int
fmt.Println(arr1)  // [0 0 0 0 0]

// 声明并初始化
arr2 := [3]int{1, 2, 3}
fmt.Println(arr2)  // [1 2 3]

// 让编译器计算长度
arr3 := [...]int{1, 2, 3, 4}
fmt.Println(arr3)  // [1 2 3 4]
```

### 数组访问

```go
arr := [3]int{10, 20, 30}

// 访问元素
fmt.Println(arr[0])  // 10

// 修改元素
arr[1] = 99
fmt.Println(arr)  // [10 99 30]

// 获取长度
fmt.Println(len(arr))  // 3
```

## <Badge text="数组长度是类型的一部分" type="warning" />

```go
// [3]int 和 [4]int 是完全不同的类型
var a1 [3]int
var a2 [4]int

// ❌ 不能赋值
// a1 = a2  // 编译错误：类型不匹配

// 只能赋值给相同长度和类型的数组
var a3 [3]int
a3 = a1  // ✅ 正确
```

<Badge text="重要" type="danger" /> 数组长度是类型的一部分，<strong>不同长度的数组是不同类型</strong>！

## <Badge text="数组初始化" type="info" />

```go
// 指定索引初始化
arr := [5]int{1: 10, 3: 30}
fmt.Println(arr)  // [0 10 0 30 0]

// 部分初始化，其余为零值
arr2 := [5]int{1, 2}
fmt.Println(arr2)  // [1 2 0 0 0]

// 使用 [...] 让编译器计算长度
arr3 := [...]string{"a", "b", "c"}
fmt.Println(len(arr3))  // 3
```

## <Badge text="数组是值类型" type="warning" />

```go
arr1 := [3]int{1, 2, 3}
arr2 := arr1  // 拷贝整个数组

arr2[0] = 99
fmt.Println(arr1)  // [1 2 3] - 原数组不受影响
fmt.Println(arr2)  // [99 2 3] - 副本被修改
```

<Badge text="注意" type="info" /> 数组赋值会<strong>拷贝整个数组</strong>，对于大数组性能开销较大。

## <Badge text="多维数组" type="info" />

```go
// 声明二维数组
var matrix [3][4]int

// 初始化二维数组
matrix2 := [2][3]int{
    {1, 2, 3},
    {4, 5, 6},
}
fmt.Println(matrix2)

// 访问元素
fmt.Println(matrix2[0][1])  // 2

// 遍历二维数组
for i := 0; i < len(matrix2); i++ {
    for j := 0; j < len(matrix2[i]); j++ {
        fmt.Printf("%d ", matrix2[i][j])
    }
    fmt.Println()
}
```

## <Badge text="数组比较" type="info" />

```go
// 数组可以比较（相同类型和长度）
a1 := [3]int{1, 2, 3}
a2 := [3]int{1, 2, 3}
a3 := [3]int{1, 2, 4}

fmt.Println(a1 == a2)  // true
fmt.Println(a1 == a3)  // false

// 不同长度的数组不能比较
// a1 == [4]int{...}  // 编译错误
```

<Badge text="限制" type="warning" /> 只有<strong>元素类型可比较</strong>的数组才能比较（如 `[]int` 不行，但 `[N]int` 可以）。

## <Badge text="数组 vs 切片" type="info" />

| 特性 | 数组 `[N]T` | 切片 `[]T` |
|-----|-----------|-----------|
| 长度 | 固定，编译时确定 | 动态，运行时变化 |
| 作为参数 | 拷贝整个数组 | 传递引用 |
| 比较 | 可以（相同长度） | 不可以 |
| 使用场景 | 固定大小数据 | 动态数据集合 |

## 练习

1. 创建一个 3x3 矩阵并计算对角线元素之和

<details>
<summary>查看答案</summary>

```go
package main

import "fmt"

func main() {
    // 创建 3x3 矩阵
    matrix := [3][3]int{
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9},
    }

    // 计算主对角线元素之和
    sumMain := 0
    for i := 0; i < 3; i++ {
        sumMain += matrix[i][i]
    }
    fmt.Printf("主对角线之和: %d\n", sumMain)  // 15 (1+5+9)

    // 计算副对角线元素之和
    sumSecondary := 0
    for i := 0; i < 3; i++ {
        sumSecondary += matrix[i][2-i]
    }
    fmt.Printf("副对角线之和: %d\n", sumSecondary)  // 15 (3+5+7)

    // 两条对角线总和
    fmt.Printf("两条对角线总和: %d\n", sumMain+sumSecondary)  // 30
}
```

**解释**：展示了二维数组的访问和对角线计算逻辑。
</details>

2. 实现数组反转函数

<details>
<summary>查看答案</summary>

```go
package main

import "fmt"

// Reverse 反转数组
func Reverse(arr [5]int) [5]int {
    length := len(arr)
    for i := 0; i < length/2; i++ {
        arr[i], arr[length-1-i] = arr[length-1-i], arr[i]
    }
    return arr
}

func main() {
    arr := [5]int{1, 2, 3, 4, 5}
    fmt.Println("原数组:", arr)  // [1 2 3 4 5]

    reversed := Reverse(arr)
    fmt.Println("反转后:", reversed)  // [5 4 3 2 1]
}
```

**解释**：使用双指针技巧原地反转数组。
</details>

3. 找出数组中的最大值和最小值

<details>
<summary>查看答案</summary>

```go
package main

import "fmt"

// FindMinMax 找出数组中的最大值和最小值
func FindMinMax(arr [5]int) (int, int) {
    if len(arr) == 0 {
        return 0, 0
    }

    min, max := arr[0], arr[0]
    for _, v := range arr[1:] {
        if v < min {
            min = v
        }
        if v > max {
            max = v
        }
    }
    return min, max
}

func main() {
    arr := [5]int{3, 1, 4, 1, 5, 9, 2, 6}
    min, max := FindMinMax(arr)

    fmt.Printf("数组: %v\n", arr)
    fmt.Printf("最小值: %d\n", min)  // 1
    fmt.Printf("最大值: %d\n", max)  // 9
}
```

**解释**：单次遍历同时找到最大值和最小值，时间复杂度 O(n)。
</details>
