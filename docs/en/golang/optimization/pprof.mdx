# pprof Usage

<Badge text="性能分析" type="danger" />

pprof 是 Go 内置的性能分析工具。

## 启用 pprof

<Badge text="初级开发者" type="tip" />

### HTTP 服务

```go
package main

import (
    "fmt"
    "net/http"
    _ "net/http/pprof"  // 导入 pprof
)

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func main() {
    // 启用 pprof HTTP 服务
    go func() {
        fmt.Println(http.ListenAndServe("localhost:6060", nil))
    }()

    // 主服务
    http.HandleFunc("/", helloHandler)
    http.ListenAndServe(":8080", nil)
}
```

<Callout type="warning">
**重要**：生产环境要限制 pprof 访问，不要暴露到公网。
</Callout>

### 基础命令

```bash
# 查看 pprof 端点
curl http://localhost:6060/debug/pprof/

# 可用端点：
# /debug/pprof/      - pprof 索引
# /debug/pprof/profile    - CPU profile
# /debug/pprof/heap        - 内存 profile
# /debug/pprof/goroutine   - goroutine profile
# /debug/pprof/block       - 阻塞 profile
```

## CPU 分析

<Badge text="中级开发者" type="warning" />

### 采集 CPU 数据

```bash
# 采集 30 秒的 CPU 数据
go tool pprof http://localhost:6060/debug/pprof/profile?seconds=30

# 或者使用 curl
curl -o cpu.prof http://localhost:6060/debug/pprof/profile?seconds=30
go tool pprof cpu.prof
```

### 分析 CPU 数据

```bash
# 进入交互模式
go tool pprof http://localhost:6060/debug/pprof/profile

# 常用命令
(pprof) top           # 查看 CPU 占用最多的函数
(pprof) top5          # 查看 top 5
(pprof) list funcName # 查看函数代码
(pprof) web           # 生成可视化图表
```

### 火焰图

```bash
# 生成火焰图
go tool pprof -http=:8080 http://localhost:6060/debug/pprof/profile

# 浏览器访问 http://localhost:8080
# 选择 "Flame Graph" 视图
```

## 内存分析

<Badge text="中级开发者" type="warning" />

### 堆分析

```bash
# 查看当前堆信息
curl http://localhost:6060/debug/pprof/heap

# 采集堆数据
go tool pprof http://localhost:6060/debug/pprof/heap

# 查看分配最多的函数
(pprof) top

# 查看特定函数的分配
(pprof) list functionName
```

### 内存泄漏检测

```bash
# 采集多次堆数据对比
go tool pprof -base=heap1.prof heap2.prof

# 查看差异
(pprof) top
```

## Goroutine 分析

<Badge text="高级开发者" type="danger" />

### 查看 Goroutine

```bash
# 查看 goroutine 堆栈
curl http://localhost:6060/debug/pprof/goroutine?debug=2

# 分析 goroutine
go tool pprof http://localhost:6060/debug/pprof/goroutine
```

## 实战案例

### 优化 HTTP 服务

```go
// 添加 pprof
import _ "net/http/pprof"

// 在代码中添加标签
import "runtime/pprof"

func heavyOperation() {
    defer pprof.StartCPUProfile(pprof.Profiles("cpu.prof"))
    defer pprof.StopCPUProfile()

    // 执行操作
}

// 使用 pprof.Label
func processRequest() {
    pprof.SetGoroutineLabels(1, "processRequest")
    defer pprof.SetGoroutineLabels(1, "")
    // ...
}
```
